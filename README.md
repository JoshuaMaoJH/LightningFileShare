# LightningFileShare
This is a project that can transfer files quickly from one computer to another computer in the same LAN.

## Features
-  **Lightning Fast** - Direct peer-to-peer transfer at 100MB/s+
-  **Zero Configuration** - Auto-discover devices on your network
-  **No Size Limit** - Transfer files of any size or type
-  **Lossless Quality** - No compression, no conversion
-  **Privacy First** - Files never leave your local network
-  **Cross-Platform** - Works on Windows, macOS, and Linux

## Quick Start

### Installation
Make sure you have Python 3+
```bash
git clone https://github.com/JoshuaMaoJH/LightningFileShare.git
cd LightningShare
```

### Run the application
```bash
python main.py
```

## Usage
1. **Connect** both devices to the **same WiFi network**
2. **Launch** the application on both devices
3. **Wait** for automatic device discovery
4. **Select** the target device from the list
5. **Send** - Click "Send File" and choose your file
6. **Receive** - Click "Receive File" and select save location
7. **Done!** Transfer completes in seconds
